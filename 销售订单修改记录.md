# 销售订单修改记录

> 来源：`沟通结果/20260509095225-转写_丁宁宇的快速会议-逐字稿文本-1.txt`  
> 整理日期：2026-05-11  
> 说明：逐字稿中提到的“订单总表 / 表格”，本次统一理解为销售订单列表页和销售订单详情页；创建页面单独按录入规则整理。

## 一、核心结论

销售订单金额字段需要区分“录入口径”和“展示口径”：

| 场景 | 口径结论 |
|---|---|
| 创建页面 / 编辑页面 | 支持按业务发生币种录入，可选择 USD、RMB 或对应本地币种；系统按汇率折算生成 RMB 金额。 |
| 销售订单列表页 | 作为汇总查看页面，金额字段默认展示 RMB，不重复展示原币金额，避免表格字段过多。 |
| 销售订单详情页 | 主展示 RMB，与列表和财务统计保持一致；同时保留原币金额、币种、汇率、折算来源作为明细或折算记录。 |
| 财务报表取数 | 统一取 RMB 折算金额作为统计口径，原币金额仅用于追溯、核对和重新计算。 |

## 二、销售订单列表页

### 2.1 页面定位

销售订单列表页用于快速查看订单核心信息和财务结果，不承担多币种录入和复杂核算说明。

### 2.2 展示规则

| 模块 | 修改规则 |
|---|---|
| 金额展示 | 列表页金额字段统一展示 RMB 折算金额。 |
| 币种展示 | 不在每个金额字段后重复展示 USD / EUR / CAD 等原币信息。 |
| 汇率信息 | 默认不在列表展示，可通过详情页查看。 |
| 表格宽度 | 避免同一字段同时展示“原币金额 + RMB金额”，防止列表过宽。 |
| 财务统计 | 列表页展示结果需与财务报表使用同一 RMB 口径。 |

### 2.3 建议展示字段

| 字段 | 列表页展示口径 | 说明 |
|---|---|---|
| 订单号 | 原始订单号 | 用于追踪订单。 |
| 平台 | 平台名称 | 如 Amazon、Temu、Wayfair 等。 |
| 店铺 | 店铺名称 | 用于店铺维度筛选和统计。 |
| 市场 | 市场名称 | 如美国、欧洲、加拿大等。 |
| SKU | SKU编码 | 支持一个订单多个 SKU 时按订单行展示。 |
| 销量 | 数量 | 展示订单商品数量。 |
| 零售价 | 原币或 RMB，按现有业务确认 | 零售价准确性要求相对低，可作为参考字段；如列表空间有限，建议不展示或仅展示 RMB。 |
| 净销售额 | RMB | 财务核心字段，列表统一展示 RMB。 |
| 采购成本 | RMB | 采购价本身以 RMB 为主。 |
| 头程运费 | RMB | 头程以 RMB 为主。 |
| 尾程运费 | RMB | 即使后台原始数据为 USD，列表也只展示 RMB。 |
| 国内直发 | RMB | 国内直发费用统一 RMB。 |
| 广告费用 | RMB | 平台后台可能为 USD，列表展示折算后的 RMB。 |
| 亚马逊仓储费 | RMB | 原始可能为 USD，列表展示 RMB。 |
| 海外仓仓储费 | RMB | 原始可能为当地币种，列表展示 RMB。 |
| 退款金额 | RMB | 创建或编辑时可选币种，列表展示 RMB。 |
| 平台补偿金 | RMB | 统一折算为 RMB。 |
| 盘点 | RMB | Amazon 相关费用或收入折算为 RMB。 |
| 清算 | RMB | Amazon 清算收入折算为 RMB。 |
| 其他费用 | RMB | 公共费用或手工费用统一 RMB。 |
| 毛利润 | RMB | 按 RMB 字段统一计算。 |

### 2.4 不建议在列表展示的字段

| 字段 | 原因 | 建议处理 |
|---|---|---|
| 原币金额 | 会导致每个金额字段重复展示，列表过宽。 | 放到订单详情页。 |
| 原币币种 | 列表页不承担核算追溯。 | 详情页展示。 |
| 汇率 | 列表页不是折算核对页面。 | 详情页展示。 |
| 汇率来源 | 对列表查询效率和阅读效率无帮助。 | 详情页或操作日志展示。 |
| 平台佣金 | 逐字稿中提到可在订单编辑中维护，但不需要在订单总表展示。 | 用于税费报表或后续核算，不进入默认列表。 |

## 三、销售订单详情页

### 3.1 页面定位

销售订单详情页用于查看单笔订单完整信息，需要同时满足业务查看、财务核对和数据追溯。

### 3.2 展示规则

| 模块 | 修改规则 |
|---|---|
| 主金额展示 | 订单详情页主展示金额统一为 RMB。 |
| 原币追溯 | 详情页需保留原币金额、原币币种、汇率、折算后 RMB。 |
| 汇率信息 | 展示汇率值、汇率日期、汇率来源，便于核对。 |
| 数据来源 | 区分系统同步、平台账单、人工录入、财务调整等来源。 |
| 修改记录 | 人工修改过的金额字段，需要记录修改人、修改时间和修改前后值。 |

### 3.3 建议页面结构

| 区块 | 展示内容 |
|---|---|
| 订单基础信息 | 订单号、平台、店铺、市场、SKU、数量、创建时间、订单状态。 |
| 财务汇总信息 | 净销售额、采购成本、各类费用、退款、补偿、毛利润，主展示 RMB。 |
| 原币折算明细 | 原币金额、原币币种、汇率、折算 RMB、汇率日期、汇率来源。 |
| 费用来源明细 | 平台账单、物流账单、仓储账单、人工录入、后台配置等来源。 |
| 操作记录 | 金额字段新增、修改、汇率重算、人工调整记录。 |

### 3.4 详情页字段口径

| 字段 | 主展示 | 明细追溯 |
|---|---|---|
| 净销售额 | RMB | 原币金额、币种、汇率、折算时间。 |
| 尾程运费 | RMB | 原始币种通常为 USD，特殊场景可为 RMB。 |
| 广告费用 | RMB | 平台后台原始币种及折算汇率。 |
| 亚马逊仓储费 | RMB | Amazon 原始金额及币种。 |
| 海外仓仓储费 | RMB | 当地币种、汇率、折算 RMB。 |
| 退款金额 | RMB | 原币退款金额、币种、汇率。 |
| 平台补偿金 | RMB | 原始补偿金额、币种、来源。 |
| 盘点 | RMB | Amazon 相关原始金额、币种、费用类型。 |
| 清算 | RMB | Amazon 清算原始金额、币种、折算 RMB。 |
| 平台佣金 | 可在详情页展示或收起 | 用于税费报表，不作为列表默认字段。 |

## 四、销售订单创建页面

### 4.1 页面定位

销售订单创建页面用于录入订单原始业务数据，需要支持按实际发生币种录入，并自动生成统一 RMB 口径。

### 4.2 录入规则

| 字段 | 创建页面录入方式 | 系统处理规则 |
|---|---|---|
| 零售价 | 可录入原币金额 | 可作为参考字段，不作为核心财务统计依据。 |
| 净销售额 | 支持选择币种录入，如 USD / RMB | 系统根据汇率生成 RMB 金额；财务报表取 RMB。 |
| 采购成本 | RMB | 采购价默认 RMB，不需要多币种。 |
| 头程运费 | RMB | 头程默认 RMB。 |
| 尾程运费 | 默认支持 USD，也允许 RMB | 系统统一折算为 RMB；订单列表和详情主展示 RMB。 |
| 国内直发 | RMB | 国内直发费用默认 RMB。 |
| 广告费用 | 支持平台原币或 RMB | 平台后台若为 USD，系统折算为 RMB；如果已经是 RMB，则直接使用。 |
| 亚马逊仓储费 | 默认 USD | 系统折算为 RMB。 |
| 海外仓仓储费 | 按海外仓当地币种录入 | 如 USD、EUR、CAD 等，系统折算为 RMB。 |
| 退款金额 | 支持 USD / RMB 等币种 | 系统折算为 RMB。 |
| 平台补偿金 | 按平台账单币种录入 | 系统折算为 RMB。 |
| 盘点 | Amazon 场景通常为 USD | 系统折算为 RMB；非 Amazon 可为空或不适用。 |
| 清算 | Amazon 场景通常为 USD | 系统折算为 RMB；非 Amazon 可为空或不适用。 |
| 其他费用 | RMB 或按配置币种 | 最终统一生成 RMB。 |
| 平台佣金 | 可在创建 / 编辑中维护 | 不进入订单列表默认展示，可供税费报表使用。 |

### 4.3 创建页面必存字段

每个涉及多币种的金额字段，建议至少保存以下数据：

| 字段 | 说明 |
|---|---|
| 原币金额 | 用户录入或平台同步的原始金额。 |
| 原币币种 | USD、RMB、EUR、CAD 等。 |
| 汇率 | 原币折算 RMB 的汇率。 |
| 汇率日期 | 使用哪一天的汇率。 |
| 汇率来源 | 系统汇率表、平台账单、人工维护等。 |
| RMB金额 | 财务报表和列表展示使用的统一金额。 |
| 数据来源 | 平台同步、人工录入、账单导入、系统计算等。 |

## 五、页面间同步影响

| 影响范围 | 需要同步修改 |
|---|---|
| 销售订单列表字段说明 | 明确列表金额统一 RMB 展示，不重复展示原币字段。 |
| 销售订单详情字段说明 | 增加“原币折算明细”或同等区域，用于展示原币、汇率和折算过程。 |
| 销售订单创建 / 编辑字段说明 | 增加币种选择、汇率折算、RMB自动生成规则。 |
| 财务报表取数规则 | 所有财务报表默认取销售订单中的 RMB 金额字段。 |
| 税费报表 | 如需要平台佣金，则从订单编辑维护字段取数，不依赖订单列表展示。 |
| 数据校验 | 原币金额、币种、汇率、RMB金额之间需要校验一致。 |
| 操作日志 | 人工修改金额、币种、汇率时记录修改前后值。 |

## 六、验收标准

| 编号 | 验收点 | 验收标准 |
|---|---|---|
| 1 | 创建页面多币种录入 | 净销售额、尾程、广告费、仓储费、退款等字段可按配置选择币种。 |
| 2 | 自动折算 RMB | 选择非 RMB 币种后，系统可根据汇率生成 RMB 金额。 |
| 3 | 列表展示一致 | 销售订单列表页金额字段统一展示 RMB，不重复展示原币金额。 |
| 4 | 详情可追溯 | 销售订单详情页可以查看原币金额、币种、汇率、折算后 RMB。 |
| 5 | 财务口径统一 | 财务报表、市场开拓简报、新品与季节简报等统计均取 RMB 金额。 |
| 6 | 平台佣金边界清晰 | 平台佣金可维护、可用于税费报表，但不进入销售订单列表默认展示。 |
| 7 | 修改有记录 | 人工调整金额、币种、汇率时，有操作记录可追溯。 |
